# Code of Conduct

## Our standard

We are committed to a respectful, technically serious, and inclusive community.

Examples of positive behavior:

- being constructive and specific
- disagreeing on ideas without attacking people
- documenting tradeoffs clearly
- helping others reproduce issues and fixes

Unacceptable behavior includes:

- harassment or personal attacks
- discriminatory language
- bad-faith disruption
- publishing private or sensitive information without permission

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- discussions
- release threads
- project communication channels tied to Scibudy

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
or other contributions that violate this Code of Conduct.

## Reporting

For conduct concerns, use the contact route documented in `SUPPORT.md`. For
security-sensitive matters, use `SECURITY.md` instead.

# 支持矩阵

## 平台支持

| 平台 | Base | Analysis | 本地 GPU 模型 |
| --- | --- | --- | --- |
| Linux | 支持 | 支持 | 支持（优先 NVIDIA） |
| macOS | 支持 | 支持 | 有限 / 延后 |
| Windows | `v0.x` 非一等目标 | `v0.x` 非一等目标 | 不支持 |

| 源码安装 | 面向贡献者支持 |

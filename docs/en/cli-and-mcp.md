# CLI and MCP

## CLI

Main commands:

- `scibudy search`
- `scibudy collect`
- `scibudy libraries`
- `scibudy ingest-library`
- `scibudy search-evidence`
- `scibudy ui`

## MCP

Scibudy exposes a local `stdio` MCP server for Codex through the managed `research` block in `~/.codex/config.toml`.

Important tool families:

- search
- open access
- library management
- context bundles
- analysis and evidence search

## Compatibility aliases

- `research-cli`
- `research-mcp`

These remain available in `v0.x` for compatibility.

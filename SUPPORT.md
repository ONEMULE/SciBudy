# Support

## Where to ask what

Use GitHub Issues for:

- reproducible bugs
- documentation problems
- install failures
- packaging or bootstrap regressions
- provider/runtime integration issues

Use Discussions if enabled, or an issue labelled as a question, for:

- usage questions
- architecture discussion
- feature proposals

## What to include

Please include:

- platform and OS version
- Python version
- install profile used
- whether you used source install or npm bootstrap
- relevant command output
- whether the problem is CPU-only or GPU/local-model related

## GPU/local-model note

Local GPU model support is Linux-first in `v0.x`. macOS support is more limited, and Windows is not yet a first-class target.

## Security note

If the issue involves credentials, secrets, or unsafe execution paths, do not file a public bug. Follow `SECURITY.md`.

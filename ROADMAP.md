# Roadmap

## v0.1.x

- stabilize install/runtime separation
- stabilize Linux local GPU retrieval path
- complete professional open-source repo organization
- improve docs and examples for external users

## v0.2.x

- publish to PyPI and npm
- add stronger CI/release validation
- improve macOS support story
- improve local model health reporting and warm-state visibility

## v0.3.x

- richer report browsing and export
- stronger semantic retrieval and reranking policies
- optional docs-site publication
- broader package/install ergonomics

## Out of scope for early v0.x

- first-class Windows support
- guaranteed multi-GPU orchestration
- heavy hosted-service deployment workflows
